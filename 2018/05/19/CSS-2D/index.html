CSS 2D 变形

Posted on 2018-05-19 | In 前端

效果如图

2D 转换

2D 变换的方法有：

  • translate()
    //translate(x,y)方法，根据左(X轴)和顶部(Y轴)位置给定的参数，从当前元素位置移动，单位：px。
  • rotate()
    //rotate()方法，在一个给定度数顺时针旋转的元素。负值是允许的，这样是元素逆时针旋转, 单位：deg。
  • scale()
    //scale(x,y)方法，该元素增加或减少的大小，取决于宽度（X轴）和高度（Y轴）的参数。
  • skew(x,y)
    //包含两个参数值，分别表示X轴和Y轴倾斜的角度，如果第二个参数为空，则默认为0，参数为负表示向相反方向倾斜,单位：deg。
  • matrix()
    // matrix( a, b, c, d, e, f );a 水平缩放,b 水平倾斜,c 垂直倾斜,d 垂直缩放,e 水平移动,f 垂直移动。

样例

HTML:

div{
width: 100px;
background:#d0e0e3;
padding: 40px 0 40px 0;
text-align: center;
float: left;
margin: 200px 50px 0 0;
font-size:20px;
}
.div1{
/ 通过 skew() 方法，元素翻转给定的角度，根据给定的水平线（X 轴）和垂直线（Y 轴）参数 /
transform: skew(30deg,0deg);
-webkit-transform: skew(30deg,0deg);
-ms-transform: skew(30deg,0deg);
-o-transform: skew(30deg,0deg);
-moz-transform: skew(30deg,0deg);

}
.div2{
/ 通过 scale() 方法，元素的尺寸会增加或减少，根据给定的宽度（X 轴）和高度（Y 轴）参数 /
transform: scale(0.5,1);
-webkit-transform: scale(0.5,1);
-ms-transform: scale(0.5,1);
-o-transform: scale(0.5,1);
-moz-transform: scale(0.5,1);
}
.div3{
/ 通过 rotate() 方法，元素顺时针旋转给定的角度。允许负值，元素将逆时针旋转。 /
transform: rotate(45deg);
-webkit-transfoem: rotate(45deg);
-ms-transform: rotate(45deg);
-moz-transform: rotate(45deg);
}
.div4{
/ 通过 translate() 方法，元素从其当前位置移动，根据给定的 left（x 坐标） 和 top（y 坐标） 位置参数 /
transform: translate(10px,20px);
-webkit-transform: translate(10px,20px);
-o-transform: translate(10px,20px);
-ms-transform: translate(10px,20px);
-moz-transform: translate(10px,20px);
}
.div5{
transform: matrix(1.2,1,0,0.8,20,60);
-webkit-transform: matrix(1.2,1,0,0.8,20,60);
-o-transform: matrix(1.2,1,0,0.8,20,60);
-ms-transform: matrix(1.2,1,0,0.8,20,60);
-moz-transform: matrix(1.2,1,0,0.8,20,60);
}
/ matrix( a, b, c, d, e, f );
a 水平缩放
b 水平倾斜
c 垂直倾斜
d 垂直缩放
e 水平移动
f 垂直移动 /
`
